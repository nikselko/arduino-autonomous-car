# Arduino-Autonomous-Car

## C++ implementation of INO Autonomous car. 

### Used lib's:

1. Arduino.h 
2. Servo.h (servo-motor lib, for steering)

### Used Hardware's:

1. Motor
2. Servo Motor
3. Arduino
4. 2 x Ultrasonic sensors
5. 3 x Infrared Sensors
